# meli_frontend

## Install Vue CLI
```
npm install -g @vue/cli
# OR
yarn global add @vue/cli

version: @vue/cli 5.0.8
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
